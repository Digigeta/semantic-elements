Semantic vs Non-semantic Elements

Semantic elements in HTML carry important information for the developer and browser.
Non-semantic elements like

  1. div
  2. span
are not included. (There's a lot more to learn about non-semantic elements, specifically "div-itis")
Semantic elements, like table and article define the content of the web page.

These tags help organize HTML structure and make websites easier to modify.
Utilizing the various elements, or tags, allows us to create dynamic web pages that:
  • display images and videos
  • provide links to other pages
  • make lists like the one you're reading now!
